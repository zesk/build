## `testTools`

> Load test tools

### Usage

    testTools [ --help ] [ binary ] [ ... ]

Load test tools

### Arguments

- `--help` - Flag. Optional. Display this help.
- `binary` - Callable. Optional. Run this program after loading test tools.
- `...` - Optional. Arguments. Arguments for binary.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
