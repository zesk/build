## `gitPreCommitExtensionList`

> List the extensions available.

### Usage

    gitPreCommitExtensionList [ --help ]

List the extensions available.

### Arguments

- `--help` - Flag. Optional. Display this help.

### Writes to standard output

String. One per line.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
