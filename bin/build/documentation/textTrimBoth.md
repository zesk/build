## `textTrimBoth`

> Trim whitespace from beginning and end of a stream

### Usage

    textTrimBoth [ --help ]

Trim whitespace from beginning and end of a stream

### Arguments

- `--help` - Flag. Optional. Display this help.

### Reads standard input

Reads lines from stdin until EOF

### Writes to standard output

Outputs modified lines

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
