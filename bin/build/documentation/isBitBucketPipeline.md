### `isBitBucketPipeline`

> Are we currently in the BitBucket pipeline?

#### Usage

    isBitBucketPipeline

Are we currently in the BitBucket pipeline?

> Location: `bin/build/tools/bitbucket.sh`

#### Arguments

- none

#### Return codes

- `0` - is BitBucket pipeline
- `1` - Not a BitBucket pipeline
