### `jsonFileGet`

> Get a value in a JSON file

#### Usage

    jsonFileGet jsonFile path

Get a value in a JSON file

> Location: `bin/build/tools/json.sh`

#### Arguments

- `jsonFile` - File. Required. File to get value from.
- `path` - String. Required. dot-separated path to get

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
