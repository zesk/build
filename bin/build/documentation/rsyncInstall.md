## `rsyncInstall`

> Install `rsync`.

### Usage

    rsyncInstall [ --help ]

Install `rsync`.
`rsync` is a tool which easily keeps file directories synchronized between
file systems, remote systems, and locations.

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
