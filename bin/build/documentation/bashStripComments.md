## `bashStripComments`

> Pipe to strip comments from a bash file

### Usage

    bashStripComments [ --help ]

Removes literally any line which begins with zero or more whitespace characters and then a `#`.

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
