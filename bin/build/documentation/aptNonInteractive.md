## `aptNonInteractive`

> Run apt-get non-interactively

### Usage

    aptNonInteractive [ ... ]

Run apt-get non-interactively

> Location: `bin/build/tools/apt.sh`

### Arguments

- `...` - Arguments. Pass through arguments to `apt-get`

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
