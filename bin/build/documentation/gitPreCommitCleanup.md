## `gitPreCommitCleanup`

> Clean up after our pre-commit (deletes cache directory)

### Usage

    gitPreCommitCleanup [ --help ]

Clean up after our pre-commit (deletes cache directory)

> Location: `bin/build/tools/git.sh`

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
