### Usage

    usageArgumentArray handler argument

Placeholder for array types

### Arguments

- `handler` - Function. Required. handler function to call upon failure.
- `argument` - String. Required. Name of the argument used in error messages.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
