### `awsProfilesList`

> List AWS profiles available in the credentials file

#### Usage

    awsProfilesList [ --help ]

List AWS profiles available in the credentials file

> Location: `bin/build/tools/aws.sh`

#### Arguments

- `--help` - Flag. Optional. Display this help.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
