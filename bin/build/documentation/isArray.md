## `isArray`

> Is a variable declared as an array?

### Usage

    isArray variableName [ --help ]

Is a variable declared as an array?

### Arguments

- `variableName` - String. Required. Variable name to check.
- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
