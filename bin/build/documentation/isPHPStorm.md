### `isPHPStorm`

> Are we within the JetBrains PHPStorm terminal?

#### Usage

    isPHPStorm [ --help ]

Are we within the JetBrains PHPStorm terminal?

> Location: `bin/build/tools/vendor.sh`

#### Arguments

- `--help` - Flag. Optional. Display this help.

#### Return codes

- `0` - within the PhpStorm terminal
- `1` - not within the PhpStorm terminal AFAIK
