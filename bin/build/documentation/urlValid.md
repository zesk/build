## `urlValid`

> Checks if a URL is valid

### Usage

    urlValid [ --help ] url ...

Checks if a URL is valid

### Arguments

- `--help` - Flag. Optional. Display this help.
- `url ...` - String. URL. Required. A Uniform Resource Locator

### Return codes

- `0` - all URLs passed in are valid
- `1` - at least one URL passed in is not a valid URL
