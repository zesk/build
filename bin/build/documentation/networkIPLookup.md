## `networkIPLookup`

> Get the current IP address of a host

### Usage

    networkIPLookup [ --help ]

Get the current IP address of a host

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
