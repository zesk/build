## `githubPublishDate`

> Output the publish date for the latest release of ownerRepository

### Usage

    githubPublishDate [ ownerRepository ]

Output the publish date for the latest release of ownerRepository

### Arguments

- `ownerRepository` - String. Github `owner/repository` string

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
