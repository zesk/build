## `isPyCharm`

> Are we within the JetBrains PyCharm terminal?

### Usage

    isPyCharm [ --help ]

Are we within the JetBrains PyCharm terminal?

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - within the PyCharm terminal
- `1` - not within the PyCharm terminal AFAIK
