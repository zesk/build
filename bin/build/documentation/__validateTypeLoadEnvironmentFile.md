### `__validateTypeLoadEnvironmentFile`

> Validates a value as an environment file which is loaded

#### Usage

    __validateTypeLoadEnvironmentFile variableValue

Validates a value as an environment file which is loaded immediately.

> Location: `bin/build/tools/validate.sh`

#### Arguments

- `variableValue` - String. Required.

#### Return codes

- `2` - Argument error
- `0` - Success
