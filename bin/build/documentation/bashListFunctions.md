## `bashListFunctions`

> List functions in a given shell file

### Usage

    bashListFunctions [ --help ] [ file ] [ --help ]

List functions in a given shell file

### Arguments

- `--help` - Flag. Optional. Display this help.
- `file` - File. Optional. File(s) to list bash functions defined within.
- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
