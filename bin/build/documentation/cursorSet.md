## `cursorSet`

> Move the cursor to x y

### Usage

    cursorSet x y

Move the cursor to x y

### Arguments

- `x` - UnsignedInteger. Required. Column to place the cursor.
- `y` - UnsignedInteger. Required. Row to place the cursor.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
