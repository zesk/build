## `isNumber`

> Test if an argument is a floating point number

### Usage

    isNumber value

Test if an argument is a floating point number
(`1e3` notation NOT supported)

### Arguments

- `value` - EmptyString. Required. Value to test.

### Return codes

- `0` - if it is a floating point number
- `1` - if it is not a floating point number
