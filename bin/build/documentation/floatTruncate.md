### `floatTruncate`

> Convert float to an integer, round down always

#### Usage

    floatTruncate [ number ]

Convert float to an integer, round down always

> Location: `bin/build/tools/float.sh`

#### Arguments

- `number` - Float. Optional. Floating point number to convert to integer.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
