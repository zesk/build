## `hostTTFB`

> Fetch Time to First Byte and other stats

### Usage

    hostTTFB [ --help ] url

Fetch Time to First Byte and other stats

### Arguments

- `--help` - Flag. Optional. Display this help.
- `url` - URL. Required. URL to check.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
