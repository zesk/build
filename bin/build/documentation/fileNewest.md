## `fileNewest`

> Output the newest file in the list

### Usage

    fileNewest file ... [ --help ]

Output the newest file in the list

### Arguments

- `file ...` - File. Required. One or more files to examine
- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
