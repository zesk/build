## `gitVersionLast`

> Get the last reported version.

### Usage

    gitVersionLast [ ignorePattern ] [ --help ]

Get the last reported version.

### Arguments

- `ignorePattern` - Optional. String. Specify a grep pattern to ignore; allows you to ignore current version
- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
