## `bashDebuggerDisable`

> Disables the debugger immediately

### Usage

    bashDebuggerDisable [ --help ]

Disables the debugger immediately
Restores file descriptors 0 1 and 2 from 20, 21 and 22 respectively

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
