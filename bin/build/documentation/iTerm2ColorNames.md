## `iTerm2ColorNames`

> Solely the color names (e.g blue), not anything else

### Usage

    iTerm2ColorNames [ --help ]

Solely the color names (e.g blue), not anything else

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
