## `pcregrepBinary`

> The name of the `pcregrep` binary on this operating system

### Usage

    pcregrepBinary [ --help ]

The name of the `pcregrep` binary on this operating system

### Arguments

- `--help` - Flag. Optional. Display this help.

### Writes to standard output

String. Name of binary for pcregrep.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
