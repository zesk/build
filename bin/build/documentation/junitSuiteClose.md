## `junitSuiteClose`

> Close tag for `testsuite`

### Usage

    junitSuiteClose [ --help ]

Close tag for `testsuite`

> Location: `bin/build/tools/junit.sh`

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
