## `stringRandom`

> Outputs 40 random hexadecimal characters, stringLowercase.

### Usage

    stringRandom

Outputs 40 random hexadecimal characters, stringLowercase.

### Arguments

- none

### Writes to standard output

`String`. A random hexadecimal string.

### Examples

    testPassword="$(stringRandom)"

### Sample Output

cf7861b50054e8c680a9552917b43ec2b9edae2b

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
