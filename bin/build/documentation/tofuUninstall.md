## `tofuUninstall`

> Uninstall tofu binary and apt sources keys

### Usage

    tofuUninstall [ package ] [ --help ]

Uninstall tofu binary and apt sources keys

### Arguments

- `package` - String. Optional. Additional packages to uninstall using `packageUninstall`
- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
