## `tofuInstall`

> Install tofu binary

### Usage

    tofuInstall [ package ] [ --help ]

Install tofu binary

### Arguments

- `package` - String. Optional. Additional packages to install using `packageInstall`
- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
