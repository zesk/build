### `yumIsInstalled`

> Is yum installed?

#### Usage

    yumIsInstalled [ --help ]

Is the yum package manager installed?

> Location: `bin/build/tools/yum.sh`

#### Arguments

- `--help` - Flag. Optional. Display this help.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
