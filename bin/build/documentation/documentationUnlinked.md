### `documentationUnlinked`

> List unlinked functions in documentation index

#### Usage

    documentationUnlinked [ --help ]

List unlinked functions in documentation index

> Location: `bin/build/tools/documentation.sh`

#### Arguments

- `--help` - Flag. Optional. Display this help.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
