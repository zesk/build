## `deployNextVersion`

> Get the next version of the supplied version

### Usage

    deployNextVersion deployHome versionName

Get the next version of the supplied version

### Arguments

- `deployHome` - Directory. Required. Deployment database home.
- `versionName` - String. Required. Application ID to look for

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
