## `mockEnvironmentStop`

> Restore a mocked value. Works solely with the default `saveGlobalName`

### Arguments

- `globalName ...` - EnvironmentVariable. Required. Global to restore from the mocked saved value.
- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
