## `linkCreate`

> Create a link

### Usage

    linkCreate [ target ] linkName

Create a link

> Location: `bin/build/tools/file.sh`

### Arguments

- `target` - Exists. File. Source file name or path.
- `linkName` - String. Required. Link short name, created next to `target`.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
