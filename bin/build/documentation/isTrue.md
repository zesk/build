## `isTrue`

> True-ish

### Usage

    isTrue [ --help ] [ value ... ]

True-ish
Succeeds when all arguments are "true"-ish

> Location: `bin/build/tools/type.sh`

### Arguments

- `--help` - Flag. Optional. Display this help.
- `value ...` - EmptyString. One or more values to test.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
