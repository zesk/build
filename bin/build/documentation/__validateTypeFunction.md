### `validate Function`

> Is a valid function currently defined?

#### Usage

    __validateTypeFunction

Is a valid function currently defined?

> Location: `bin/build/install.sample.sh`

#### Arguments

- none

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
