## `timingFormat`

> Format a timing output (milliseconds) as seconds using a decimal

### Usage

    timingFormat [ --help ] [ delta ]

Format a timing output (milliseconds) as seconds using a decimal

### Arguments

- `--help` - Flag. Optional. Display this help.
- `delta` - Integer. Milliseconds

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
