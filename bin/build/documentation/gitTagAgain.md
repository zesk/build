## `gitTagAgain`

> Remove a tag everywhere and tag again on the current

### Usage

    gitTagAgain [ tag ]

Remove a tag everywhere and tag again on the current branch

> Location: `bin/build/tools/git.sh`

### Arguments

- `tag` - String. Optional. The tag to tag again.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
