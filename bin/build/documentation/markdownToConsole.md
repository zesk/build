## `markdownToConsole`

> Converts backticks, bold and italic to console colors.

### Usage

    markdownToConsole [ --help ]

Converts backticks, bold and italic to console colors.

### Arguments

- `--help` - Flag. Optional. Display this help.

### Reads standard input

Markdown

### Writes to standard output

decorated console output

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
