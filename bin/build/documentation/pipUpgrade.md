## `pipUpgrade`

> Utility to upgrade pip correctly

### Usage

    pipUpgrade [ --help ] [ --bin binary ]

Utility to upgrade pip correctly

### Arguments

- `--help` - Flag. Optional. Display this help.
- `--bin binary` - Executable. Optional. Binary for `pip`.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
