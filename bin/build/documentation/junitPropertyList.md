## `junitPropertyList`

> Output list of `property` tags

### Usage

    junitPropertyList [ --help ] [ nameValue ... ]

Output list of `property` tags

### Arguments

- `--help` - Flag. Optional. Display this help.
- `nameValue ...` - Optional. String. A list of name value pairs (unquoted) to output as XML `property` tags.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
