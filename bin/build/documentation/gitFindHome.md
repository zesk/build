### `gitFindHome`

> Find `git` home directory

#### Usage

    gitFindHome [ startingDirectory ]

Finds `.git` directory above or at `startingDirectory`.

> Location: `bin/build/tools/git.sh`

#### Arguments

- `startingDirectory` - Directory. Optional.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
