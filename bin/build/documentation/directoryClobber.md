### `directoryClobber`

> Copy directory over another sort-of-atomically

#### Usage

    directoryClobber source target

Copy directory over another sort-of-atomically

> Location: `bin/build/tools/directory.sh`

#### Arguments

- `source` - Directory. Required. target
- `target` - FileDirectory. Required.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
