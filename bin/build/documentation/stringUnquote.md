### `stringUnquote`

> Unquote a string

#### Usage

    stringUnquote quote value

Unquote a string

> Location: `bin/build/tools/quote.sh`

#### Arguments

- `quote` - String. Required. Must match beginning and end of string.
- `value` - String. Required. Value to unquote.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
