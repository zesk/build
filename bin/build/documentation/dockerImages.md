### `dockerImages`

> List docker images which are currently pulled

#### Usage

    dockerImages [ --filter reference ]

List docker images which are currently pulled

> Location: `bin/build/tools/docker.sh`

#### Arguments

- `--filter reference` - String. Optional. Filter list by reference provided.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
