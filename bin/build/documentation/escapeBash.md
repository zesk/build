## `escapeBash`

> Converts strings to shell escaped strings

### Usage

    escapeBash [ string ]

Converts strings to shell escaped strings

### Arguments

- `string` - String. Optional. String to convert to a bash-compatible string.

### Reads standard input

text - Optional.

### Writes to standard output

bash-compatible string

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
