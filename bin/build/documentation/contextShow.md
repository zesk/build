## `contextShow`

> Show the current editor being used as a text string

### Usage

    contextShow

Show the current editor being used as a text string

### Arguments

- none

### Return codes

- `1` - If no editor or running program can be determined

### Environment

- {SEE:EDITOR.sh} - Used as a default editor (first)
- {SEE:VISUAL.sh} - Used as another default editor (last)
