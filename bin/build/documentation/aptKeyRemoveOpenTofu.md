## `aptKeyRemoveOpenTofu`

> Remove keys to disable apt to download tofu from opentofu.org

### Usage

    aptKeyRemoveOpenTofu [ --help ]

Remove keys to disable apt to download tofu from opentofu.org

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `1` - Environment problems
- `0` - All good to install tofu
