### `bashCoverageEnabled`

> Is bash coverage currently running?

#### Usage

    bashCoverageEnabled [ --help ]

Is bash coverage currently running?

> Location: `bin/build/tools/coverage.sh`

#### Arguments

- `--help` - Flag. Optional. Display this help.

#### Return codes

- `0` - Yes, it's running.
- `1` - No, it is not running.=
- `2` - Argument error
