### `__usageMessageIcon`

> Icon for usage messages

#### Usage

    __usageMessageIcon

- `0` - meaning no error, icon is `🏆`
- non-`0` - Error, icon is `❌`

> Location: `bin/build/install.sample.sh`

#### Arguments

- none

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
