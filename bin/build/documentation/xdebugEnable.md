## `xdebugEnable`

> Enable Xdebug on systems that have it

### Usage

    xdebugEnable [ --help ]

Enable Xdebug on systems that have it

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
