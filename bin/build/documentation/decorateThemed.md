## `decorateThemed`

> Applies the current theme to text rendered using `decorateThemelessMode`

### Usage

    decorateThemed [ --help ]

Applies the current theme to text rendered using `decorateThemelessMode`

### Arguments

- `--help` - Flag. Optional. Display this help.

### Reads standard input

Text to apply current theme to

### Writes to standard output

Console-ready text

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
