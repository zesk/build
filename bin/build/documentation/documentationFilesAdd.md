### `documentationFilesAdd`

> Git add documentation files

#### Usage

    documentationFilesAdd [ --help ]

Git add documentation files
Just the first path.

> Location: `bin/build/tools/documentation.sh`

#### Arguments

- `--help` - Flag. Optional. Display this help.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
