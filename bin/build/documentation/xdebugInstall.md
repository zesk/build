### `xdebugInstall`

> Install the xdebug PHP Debugger

#### Usage

    xdebugInstall [ --help ]

Install the xdebug PHP Debugger using `pear` and `pecl`.

> Location: `bin/build/tools/xdebug.sh`

#### Arguments

- `--help` - Flag. Optional. Display this help.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
