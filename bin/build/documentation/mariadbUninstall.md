## `mariadbUninstall`

> Uninstall `mariadb`

### Usage

    mariadbUninstall

Uninstall `mariadb`
When this tool succeeds the `mariadb` binary will no longer be available in the local operating system.

### Arguments

- none

### Return codes

- `1` - If uninstallation fails
- `0` - If uninstallation succeeds
