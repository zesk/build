## `__xmlAttributeValue`

> Output an XML attribute value with proper quoting

### Usage

    __xmlAttributeValue [ value ]

Output an XML attribute value with proper quoting

### Arguments

- `value` - EmptyString. Optional.

### Writes to standard output

String. XML attribute value

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
