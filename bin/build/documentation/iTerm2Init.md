## `iTerm2Init`

> Add iTerm2 support to console

### Usage

    iTerm2Init [ --ignore | -i ]

Add iTerm2 support to console

### Arguments

--ignore |- ` -i` - Flag. Optional. If the current terminal is not iTerm2, then exit status 0 and do nothing.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
