## `gitPreCommitHeader`

> Output a display for pre-commit files changed.

### Usage

    gitPreCommitHeader [ extension ] [ --help ]

Output a display for pre-commit files changed.

### Arguments

- `extension` - String. Optional. Extension to display
- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
