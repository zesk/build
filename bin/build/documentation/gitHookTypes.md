## `gitHookTypes`

> List current valid git hook types

### Usage

    gitHookTypes [ --help ]

List current valid git hook types
Hook types:
- `pre-commit`
- `pre-push`
- `pre-merge-commit`
- `pre-rebase`
- `pre-receive`
- `update`
- `post-update`
- `post-commit`

### Arguments

- `--help` - Flag. Optional. Display this help.

### Sample Output

lines:gitHookType

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
