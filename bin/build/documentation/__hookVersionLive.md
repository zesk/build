## `__hookVersionLive`

> Fetch the current live version of the software using GitHub

### Usage

    __hookVersionLive

Fetch the current live version of the software using GitHub APIs

### Arguments

- none

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error

### Environment

- {SEE:GITHUB_REPOSITORY_OWNER.sh} - Repository owner to fetch
- {SEE:GITHUB_REPOSITORY_NAME.sh} - Repository name to fetch
