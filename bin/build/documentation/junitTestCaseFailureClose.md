## `junitTestCaseFailureClose`

> Close tag for `failure` - test failed

### Usage

    junitTestCaseFailureClose [ --help ]

Close tag for `failure` - test failed

> Location: `bin/build/tools/junit.sh`

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
