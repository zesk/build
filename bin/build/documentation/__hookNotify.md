### Arguments

- `--sound soundName` - Play a sound
- `--title title` - Set the title of the notification
- `message ...` - Display this message (one per line) in the notification.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error

### Environment

- {SEE:BUILD_NOTIFY_SOUND.sh} - Play this sound if none is specified.
