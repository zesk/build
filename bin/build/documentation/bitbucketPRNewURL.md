### `bitbucketPRNewURL`

> Compute the URL to create a new PR

#### Usage

    bitbucketPRNewURL [ organization ] [ repository ]

Compute the URL to create a new PR

> Location: `bin/build/tools/bitbucket.sh`

#### Arguments

- `organization` - String. Organization name.
- `repository` - String. Repository name.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
