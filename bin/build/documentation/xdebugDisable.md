## `xdebugDisable`

> Disable Xdebug on systems that have it

### Usage

    xdebugDisable [ --help ]

Disable Xdebug on systems that have it

> Location: `bin/build/tools/xdebug.sh`

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
