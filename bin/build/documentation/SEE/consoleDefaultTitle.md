## `consoleDefaultTitle`

> Set the title of the window for the console to

### Usage

    consoleDefaultTitle [ None ]

Set the title of the window for the console to "user@hostname: pwd"

> Location: `bin/build/tools/console.sh`

### Arguments

- None

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
