## `packageManagerDefault`

> Determine the default package manager on this platform.

### Usage

    packageManagerDefault

Determine the default package manager on this platform.
Output is one of:
- apk apt brew port

> Location: `bin/build/tools/package.sh`

### Arguments

- none

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
