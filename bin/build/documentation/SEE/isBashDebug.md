## `isBashDebug`

> Returns whether the shell has the debugging flag set

### Usage

    isBashDebug

Returns whether the shell has the debugging flag set

Useful if you need to temporarily enable or disable it.

> Location: `bin/build/tools/debug.sh`

### Arguments

- none

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
