## `decorations`

> Output a list of build-in decoration styles, one per line

### Usage

    decorations [ --help ]

Output a list of build-in decoration styles, one per line

> Location: `bin/build/tools/decorate/core.sh`

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
