## `networkNameFull`

> Platform-agnostic host name

### Usage

    networkNameFull

Get the full hostname on the current platform.
Formerly `hostname``Full`.

> Location: `bin/build/tools/host.sh`

### Arguments

- none

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
