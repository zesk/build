## `junitTestCaseErrorOpen`

> Open tag for `error` - test error output

### Usage

    junitTestCaseErrorOpen [ --help ]

Open tag for `error` - test error output

> Location: `bin/build/tools/junit.sh`

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
