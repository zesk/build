## `buildDebugStop`

> Stop build debugging if it is enabled

### Usage

    buildDebugStop [ --help ]

Stop build debugging if it is enabled

> Location: `bin/build/tools/debug.sh`

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error

- {SEE:buildDebugEnabled}
