## `bashDebuggerEnable`

> Enable the debugger

### Usage

    bashDebuggerEnable [ --help ]

Enables the debugger immediately.

Saves file descriptors 0 1 and 2 as 20, 21 and 22 respectively

> Location: `bin/build/tools/debugger.sh`

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
