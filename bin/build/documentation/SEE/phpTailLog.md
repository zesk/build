## `phpTailLog`

> Tail the PHP log

### Usage

    phpTailLog [ --help ]

Tail the PHP log

> Location: `bin/build/tools/php.sh`

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
