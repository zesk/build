## `decorations`

> Output a list of build-in decoration styles, one per line

### Usage

    decorations [ --help ]

Output a list of build-in decoration styles, one per line

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error

### Requires

__help convertValue
