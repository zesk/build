### `awsInstall`

> aws Command-Line install

#### Usage

    awsInstall

aws Command-Line install

Installs x86 or aarch64 binary based on `HOSTTYPE`.

> Location: `bin/build/tools/aws.sh`

#### Arguments

- none

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
