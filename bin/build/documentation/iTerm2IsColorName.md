### `iTerm2IsColorName`

> Is it a color name?

#### Usage

    iTerm2IsColorName [ --help ]

Is it a color name?

> Location: `bin/build/tools/iterm2.sh`

#### Arguments

- `--help` - Flag. Optional. Display this help.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
