## `pathShow`

> Show the path and where binaries are found

### Usage

    pathShow [ --help ] [ binary ]

Show the path and where binaries are found

### Arguments

- `--help` - Flag. Optional. Display this help.
- `binary` - Executable. Optional. Display where this executable appears in the path.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
