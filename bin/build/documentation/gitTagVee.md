## `gitTagVee`

> Given a tag in the form "1.1.3" convert it to

### Usage

    gitTagVee

Given a tag in the form "1.1.3" convert it to "v1.1.3" so it has a character prefix "v"
Delete the old tag as well

### Arguments

- none

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
