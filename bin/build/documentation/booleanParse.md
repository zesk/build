## `booleanParse`

> Parses text and determines if it's true-ish

### Usage

    booleanParse

Parses text and determines if it's true-ish
Without arguments, displays help.

### Arguments

- none

### Return codes

- `0` - true
- `1` - false
- `2` - Neither
- `0` - Text is plain
- `1` - Text contains non-plain characters

### Requires

stringLowercase __help
