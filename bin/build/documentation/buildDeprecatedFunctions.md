## `buildDeprecatedFunctions`

> List all functions which are currently deprecated in Zesk Build

### Usage

    buildDeprecatedFunctions [ --help ]

List all functions which are currently deprecated in Zesk Build

### Arguments

- `--help` - Flag. Optional. Display this help.

### Writes to standard output

String

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
