## `isInteger`

> Test if an argument is a signed integer

### Usage

    isInteger value [ value ]

Test if an argument is a signed integer

### Arguments

- `value` - EmptyString. Required. Value to test.
- `value` - EmptyString. The value to test.

### Return codes

- `0` - if it is a signed integer
- `1` - if it is not a signed integer
