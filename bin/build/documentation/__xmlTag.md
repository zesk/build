## `__xmlTag`

> Output am XML single tag

### Usage

    __xmlTag name [ ... ]

Output am XML single tag

### Arguments

- `name` - String. Required. Tag name.
- `...` - Arguments. Optional. Attributes to output within the tag in name/value pairs as `name=value`

### Writes to standard output

String. XML single tag

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
