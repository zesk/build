### `hookEnvironment`

> Load hook-related environment variables

#### Usage

    hookEnvironment [ --help ]

Load hook environment variables used to find hooks

> Location: `bin/build/tools/hook.sh`

#### Arguments

- `--help` - Flag. Optional. Display this help.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
