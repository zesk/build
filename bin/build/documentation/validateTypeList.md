### `validateTypeList`

> Types which can be validated

#### Usage

    validateTypeList [ --help ]

List types which can be validated.

> Location: `bin/build/tools/validate.sh`

#### Arguments

- `--help` - Flag. Optional. Display this help.

#### Writes to standard output

Type

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
