## `manPathCleanDuplicates`

> Cleans the MANPATH and removes non-directory entries and duplicates

### Usage

    manPathCleanDuplicates [ --help ]

Cleans the MANPATH and removes non-directory entries and duplicates
Maintains ordering.

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
