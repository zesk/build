## `decorateStyle`

> Fetch

### Usage

    decorateStyle style [ newFormat ]

Fetch

### Arguments

- `style` - String. Required. The style to fetch or replace.
- `newFormat` - String. Optional. The new style formatting options as a string in the form `lp dp label`

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
