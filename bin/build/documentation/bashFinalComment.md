## `bashFinalComment`

> Extracts the final comment from a stream

### Usage

    bashFinalComment [ --help ]

Extracts the final comment from a stream

### Arguments

- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error

### Requires

fileReverseLines sed cut grep convertValue
