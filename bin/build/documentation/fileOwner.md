## `fileOwner`

> Get the file owner name

### Usage

    fileOwner [ file ] [ --help ]

Get the file owner name
Outputs the file owner for each file passed on the command line

### Arguments

- `file` - File to get the owner for
- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `1` - Unable to access file
