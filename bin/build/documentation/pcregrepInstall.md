### `pcregrepInstall`

> Install pcregrep binary

#### Usage

    pcregrepInstall [ --help ]

Install pcregrep binary

> Location: `bin/build/tools/pcregrep.sh`

#### Arguments

- `--help` - Flag. Optional. Display this help.

#### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
