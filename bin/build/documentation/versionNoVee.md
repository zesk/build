## `versionNoVee`

> Take one or more versions and strip the leading `v`

### Usage

    versionNoVee

Take one or more versions and strip the leading `v`

### Arguments

- none

### Reads standard input

Versions containing a preceding `v` character (optionally)

### Writes to standard output

Versions with the initial `v` (if it exists) removed

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
