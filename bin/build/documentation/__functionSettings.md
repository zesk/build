## `__functionSettings`

> Load cached function comment values

### Usage

    __functionSettings [ home ] [ functionName ]

Requires:

### Arguments

- `home` - Directory. BUILD_HOME
- `functionName` - String. Function to fetch settings for

### Return codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
