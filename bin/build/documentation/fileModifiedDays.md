## `fileModifiedDays`

> Prints days (integer) since modified

### Usage

    fileModifiedDays file ... [ --help ]

Prints days (integer) since modified

### Arguments

- `file ...` - File. Required. One or more files to examine
- `--help` - Flag. Optional. Display this help.

### Return codes

- `0` - Success
- `2` - Can not get modification time
