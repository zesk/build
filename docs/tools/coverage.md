### `bashCoverage` - Collect code coverage statistics for a code sample

Collect code coverage statistics for a code sample
Covert resulting files using `bashCoverageReport`

- Location: `bin/build/tools/coverage.sh`

#### Usage

_mapEnvironment

#### Arguments

- No arguments.

#### Exit codes

- `0` - Success
- `1` - Environment error
- `2` - Argument error
